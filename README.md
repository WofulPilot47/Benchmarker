# PC Benchmarking Application

This application benchmarks various components of your PC including CPU, GPU, RAM, and Disk.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/WofulPilot47/benchmarker.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the application:
```bash
python main.py
```
